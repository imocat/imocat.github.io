隱私政策

最後更新：2026年1月19日

1. 我們收集的資訊

KeyAura 旨在尊重您的隱私。我們僅收集應用程式正常運行所需的最低限度的資訊：

1.1 鍵盤使用資料

  • 按鍵計數：鍵盤上每個按鍵被按下的次數
  • 時間戳：按鍵發生的時間（用於每日資料組織）
  • 鍵盤佈局：您選擇的鍵盤佈局偏好
  • 應用設置：顏色方案、熱力圖樣式和顯示偏好

1.2 系統資訊

  • 輔助功能權限：您是否已授予輔助功能權限（鍵盤監控所需）
  • 許可證狀態：您使用的是免費版還是專業版

2. 資料存儲

2.1 僅限本地存儲

KeyAura 收集的所有資料都僅存儲在您的 Mac 本地。我們不會上傳、傳輸或存儲任何資料到我們的伺服器。

2.2 存儲位置

您的資料存儲在：

  • 位置： ~/Library/Application Support/KeyAura/
  • 格式： SQLite 資料庫
  • 加密： 資料以明文形式存儲（建議加密您的 Mac 以獲得額外安全性）

2.3 資料保留

KeyAura 將無限期保留您的鍵盤使用資料，直到您選擇刪除它。您可以隨時通過應用設置刪除所有資料。

3. 我們不收集的資料

KeyAura 不收集：

  • ✗ 您輸入的實際字符（我們只統計按鍵次數）
  • ✗ 密碼、個人資訊或敏感資料
  • ✗ 屏幕截圖或屏幕錄製
  • ✗ 應用使用記錄或瀏覽歷史
  • ✗ 位置資料或設備標識符
  • ✗ 通過互聯網傳輸的任何資料

4. 我們如何使用您的資料

我們僅將您的資料用於：

  • ✓ 生成鍵盤熱力圖可視化
  • ✓ 顯示使用統計資訊（總按鍵數、最常用按鍵）
  • ✓ 提供歷史資料分析（每日、每週、每月、每年）
  • ✓ 保存您的應用偏好設置
  • ✓ 管理您的許可證狀態（免費版 vs 專業版）

5. 第三方服務

5.1 Apple App Store

如果您從 Mac App Store 下載 KeyAura，Apple 可能會收集：

  • 您的 Apple ID 和電子郵件地址
  • 設備資訊
  • 應用使用統計資訊

詳情請參閱 Apple 的 隱私政策。

5.2 Apple StoreKit（專業版）

如果您購買 KeyAura 專業版，Apple 可能會收集：

  • 購買交易資料
  • 支付資訊
  • 收據驗證資料

詳情請參閱 Apple 的 隱私政策。

6. 資料共享

KeyAura 不會與任何第三方共享您的資料。我們不出售、出租或交易您的個人資訊。

7. 您的權利

您對您的資料擁有以下權利：

  • 訪問： 通過應用界面查看所有收集的資料
  • 刪除： 隨時永久刪除所有鍵盤使用資料
  • 導出： 將資料導出為 JSON 供個人使用
  • 選擇退出： 隨時禁用鍵盤監控

8. 兒童隱私

KeyAura 不面向 13 歲以下的兒童。我們不會故意收集兒童的個人資訊。

9. 安全性

我們採取合理的安全措施來保護您的資料：

  • ✓ 所有資料存儲在您的設備本地
  • ✓ 不通過互聯網傳輸按鍵資料
  • ✓ 定期更新應用程式以解決安全漏洞

但是，沒有一種互聯網傳輸方法是 100% 安全的。我們無法保證絕對的安全。

10. 政策變更

我們可能會不時更新此隱私政策。我們將通過以下方式通知您任何更改：

  • 在此頁面上發布新政策
  • 更新"最後更新"日期
  • 通過應用更新通知用戶（如適用）

11. 輔助功能權限

KeyAura 需要輔助功能權限才能運行。此權限允許應用程式：

  • 監控全局鍵盤事件
  • 統計按鍵次數
  • 識別正在按下哪些按鍵

重要： KeyAura 不會讀取您輸入的實際字符。它只統計哪些按鍵被按下以及按下的頻率。

13. 許可協議

使用 KeyAura 即表示您同意此隱私政策。如果您不同意此政策，請不要使用該應用程式。
