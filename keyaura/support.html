Frequently Asked Questions

How do I grant keyboard monitoring permissions?
KeyAura requires Input Monitoring permission to track your keyboard usage. Here's how to grant it:

1. Open System Settings from the Apple menu
2. Go to Privacy & Security
3. Find "Input Monitoring" in the list
4. Toggle the switch next to KeyAura to enable it
5. Restart KeyAura for the changes to take effect

Note: This permission is only used to count key presses. KeyAura does not read the actual characters you type.
Why does KeyAura need keyboard monitoring permissions?
KeyAura needs Input Monitoring permission to detect which keys are being pressed on your keyboard. This is necessary for generating the heatmap visualization and statistics.

Important: KeyAura only counts key presses and does NOT record the actual characters you type. Your passwords and sensitive information remain private.
Is my keyboard data uploaded to the cloud?
No. All keyboard usage data is stored locally on your Mac in the Application Support folder. KeyAura does not upload, transmit, or store any data on external servers. Your privacy is protected.
What's the difference between Core and Pro versions?
Core (Free):
- Real-time keyboard heatmap
- Today's data statistics only
- 3 color themes
- Basic keyboard layouts
- Export with watermark

Pro ($9.99 one-time):
- All Core features
- Full time range statistics (week, month, year, custom)
- All 10 color themes
- All 4 keyboard layouts
- Export without watermark
- Export JSON data
- Priority support
How do I export my heatmap as an image?
1. Open KeyAura and make sure your heatmap is displayed
2. Click the "Share/Export" button in the top toolbar
3. Select "Export Image"
4. Choose a location to save the file
5. The image will be saved as a high-resolution PNG file

Pro users can export without watermark. Core version exports will include a small watermark.
How do I reset all my keyboard data?
1. Open KeyAura
2. Click the "Settings" button in the top toolbar
3. Scroll down to the "Data Management" section
4. Click "Reset All Data"
5. Confirm the action

Warning: This will permanently delete all keyboard usage data. This action cannot be undone.
Which macOS versions are supported?
KeyAura requires macOS 13.0 (Ventura) or later. It is optimized for Apple Silicon (M1/M2/M3) Macs but also works on Intel-based Macs.
How do I change the keyboard layout?
1. Open KeyAura
2. Click the "Settings" button in the top toolbar
3. Find the "Keyboard Layout" section
4. Choose from: 104%, 87%, 84%, 68% layouts
5. The heatmap will automatically update to match your selected layout
Can I use KeyAura on multiple Macs?
Yes, you can download and use KeyAura on multiple Macs. However, each Mac will have its own separate keyboard usage data stored locally. If you purchase the Pro version, you can restore your purchase on each Mac through the App Store.
How do I restore my Pro purchase?
If you've already purchased the Pro version:

1. Make sure you're signed in to the App Store with the same Apple ID used for purchase
2. Open KeyAura
3. Go to Settings → Pro Upgrade
4. Click "Restore Purchase"
5. Your Pro features should be unlocked automatically
